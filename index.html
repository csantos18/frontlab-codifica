Sobre o projeto

Uma pagina simples, objetiva e funcional.

A Trilha Front-End simula uma pagina de inscricao para um curso de programacao web. O objetivo e demonstrar estrutura HTML, estilo visual com CSS, interacoes com JavaScript e integracao com uma API publica.

01

HTML semantico

Estrutura organizada com cabecalho, secoes principais e formulario acessivel.

02

CSS responsivo

Layout adaptado para celular e desktop usando grid, flexbox e variaveis de cor.

03

JavaScript pratico

Interacao no CTA, validacao simples do formulario e consulta de CEP com ViaCEP.

Inscricao

Preencha seus dados para receber informacoes.

Digite um CEP valido para buscar endereco automaticamente pela API ViaCEP. Os dados nao sao enviados para servidor; a acao acontece apenas no navegador.

Endereco consultado Informe um CEP para consultar a API ViaCEP.
