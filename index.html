Sobre o projeto

Uma entrega academica simples, responsiva e funcional.

A pagina apresenta uma solucao profissional ficticia, mas funcional como entrega academica: estrutura HTML, estilo responsivo com CSS, interacoes com JavaScript e integracao com API publica.

01

HTML semantico

Cabecalho, secoes, formulario e rodape organizados para leitura clara.

02

CSS responsivo

Layout adaptado para celular, tablet e desktop com visual moderno.

03

API ViaCEP

Consulta de endereco por CEP integrada ao formulario de contato.

Tecnologias

Base simples, moderna e coerente com a atividade.

HTML5 CSS3 JavaScript Fetch API ViaCEP Git GitHub

JavaScript

Interacoes implementadas no navegador.

  • Validacao do formulario de contato.
  • Consulta automatica de CEP usando Fetch API.
  • Formatacao do campo de CEP durante a digitacao.
  • Mensagens dinamicas de erro e sucesso.
  • Menu responsivo para telas menores.
